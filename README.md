# RemoteMonster iOS SDK

RemoteMonster - Livecast Management in the Cloud

* [Website](https://remotemonster.com)

## Get SDK

### Package Manager - Cocoapods

```ruby
# Podfile
use_frameworks!
target 'MyApp' do
  pod 'RemoteMonster', '~> 2.6.1'
  # pod 'RemoteMonster', '~> 2.4.498' #if you use swift 4 
end
```

```shell
# shell

pod install
```
