# RemoteMonster iOS SDK

RemoteMonster - Livecast Management in the Cloud

* [Website](https://remotemonster.com)

## Get SDK

### Package Manager - Cocoapods 1.9.0 or higher

```ruby
# Podfile
use_frameworks!
target 'MyApp' do
pod 'RemoteMonster', '~> 2.7.0'
end
```

```shell
# shell

pod install
```
